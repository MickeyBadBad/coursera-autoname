Coursera-AutoName
=================

Auto Rename Coursera Video And Srt Filename Download By IDM(Internet Download Manager) ! 

Getting Started
=================

Download the main.py to your Coursera Video And srt path , File Tree Like:



    $Folder
    $ |--main.py
    $ |--Video File
    $ |--srt File

And run:

    $ python main.py
or
    click to run
